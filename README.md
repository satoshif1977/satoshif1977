## Development Style

**Claude Code** と対話しながら、AWS のベストプラクティスに沿った構成を形にしています。
Terraform を軸に、インフラ構築・コーディング・ドキュメント整備まで AI を活用して効率的に進めることを重視しています。
「まだ実務経験は浅い」からこそ、AI ツールを最大限に使いながら質の高いアウトプットを出す方法を日々探っています。
